## \ud83d\udcd8 Verse-Level Fields (`fields`)

These fields can be used in the `fields` query parameter:

| Field | Description |
|-------|-------------|
| `chapter_id` | Chapter number of this verse |
| `text_indopak` | \u2014 |
| `text_imlaei_simple` | \u2014 |
| `text_imlaei` | Word text in simple/Imlaei script |
| `text_uthmani` | Word text in Uthmanic script |
| `text_uthmani_simple` | \u2014 |
| `text_uthmani_tajweed` | \u2014 |
| `text_qpc_hafs` | \u2014 |
| `qpc_uthmani_hafs` | \u2014 |
