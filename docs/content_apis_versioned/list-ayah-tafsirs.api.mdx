## Get tafsirs for specific Ayah

Returns the tafsīr text for a single ayah from a specific tafsīr resource.

---

### HTTP Request

```text
GET /tafsirs/{resource_id}/by_ayah/{ayah_key}
```

| Path parameter | Type    | Required | Description                                                                                         |
|----------------|---------|----------|-----------------------------------------------------------------------------------------------------|
| `resource_id`  | number  | ✔︎       | ID of the tafsīr resource (see `/resources` endpoint).                                              |
| `ayah_key`     | string  | ✔︎       | Surah-and-ayah key, e.g.&nbsp;`2:255` (surah 2, ayah 255).                                          |

---

#### Schema

```jsonc
{
  "tafsir": {
    "verses": {
      "«ayah_key»": { "id": /* integer */ }
    },
    "resource_id":   /* integer */,
    "resource_name": /* string  */,
    "language_id":   /* integer */,
    "slug":          /* string  */,
    "translated_name": {
      "name":          /* string */,
      "language_name": /* string */
    },
    "text": /* string — HTML-formatted body */
  }
}
```
